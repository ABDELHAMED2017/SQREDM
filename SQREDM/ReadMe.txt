
To implement this solver, please 

[1] run startup.m first to add the path;

[2] run demonSNL.m to solve sensor network localization problems; or

[3] run demonMC.m to solve molecular conformation problems;



This source code contains the algorithm described in:

[1] S.L. Zhou, N.H. Xiu and H.-D. Qi, 
"A Fast Matrix Majorization-Projection Method for 
Penalized Stress Minimization with Box Constraints", 2017.

Please give credits to this paper if you use the code for your research.
